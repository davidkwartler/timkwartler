# timkwartler.com

Portfolio website built for Tim Kwartler.

**Live:** [www.timkwartler.com](https://www.timkwartler.com)

## Features

- Animated aurora gradient hero built with Motion
- Career timeline with experience highlights
- Full resume with skills and education
- Responsive design with mobile navigation

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Motion (Framer Motion)
- Deployed on Vercel
